### РҶРҪС–СҶС–Р°Р»С–Р·Р°СҶС–СҸ

```
main()
  в””в”Җв”Җ Engine::init()
      в”ңв”Җв”Җ Window::init() в”Җв”Җ> GLFW window + Vulkan surface
      в”ңв”Җв”Җ Instance::init() в”Җв”Җ> VkInstance
      в”ңв”Җв”Җ Device::init() в”Җв”Җ> GPU selection + VkDevice
      в”ңв”Җв”Җ Swapchain::init() в”Җв”Җ> VkSwapchainKHR + image views
      в”ңв”Җв”Җ DepthBuffer::init() в”Җв”Җ> Z-buffer РҙР»СҸ depth testing
      в”ңв”Җв”Җ RenderPass::init() в”Җв”Җ> VkRenderPass (color + depth)
      в”ңв”Җв”Җ Framebuffer::init() в”Җв”Җ> VkFramebuffer[] РҙР»СҸ РәРҫР¶РҪРҫРіРҫ image
      в”ңв”Җв”Җ CommandPool::init() в”Җв”Җ> VkCommandPool + VkCommandBuffer[]
      в”ңв”Җв”Җ FrameSync::init() в”Җв”Җ> Semaphores + Fences
      в”ңв”Җв”Җ Pipeline::init() в”Җв”Җ> 3D voxel pipeline
      в”ңв”Җв”Җ Pipeline2D::init() в”Җв”Җ> 2D UI pipeline
      в”ңв”Җв”Җ UIRenderer::init() в”Җв”Җ> TextRenderer + UI shaders
      в”ңв”Җв”Җ WorldRenderer::init() в”Җв”Җ> Voxel rendering setup
      в””в”Җв”Җ StateManager в”Җв”Җ> Register states (MainMenu, Playing, etc.)
```

## Memory Management

### Vulkan Object Lifecycle

Р’СҒС– Vulkan РҫРұРіРҫСҖСӮРәРё РІРёРәРҫСҖРёСҒСӮРҫРІСғСҺСӮСҢ RAII:

```cpp
class Buffer {
    Buffer() = default;
    bool init(Device& d, VkDeviceSize s, ...) {
        vkCreateBuffer(device, &info, nullptr, &buffer);
        vkAllocateMemory(device, &allocInfo, nullptr, &memory);
    }
    ~Buffer() {
        vkDestroyBuffer(device, buffer, nullptr);
        vkFreeMemory(device, memory, nullptr);
    }
};
```
