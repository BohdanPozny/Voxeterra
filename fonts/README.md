# Fonts Directory

Помісти сюди TTF шрифт з назвою `default.ttf`

## Де взяти безкоштовний шрифт:

1. **Roboto** - https://fonts.google.com/specimen/Roboto
2. **Open Sans** - https://fonts.google.com/specimen/Open+Sans
3. **Ubuntu Mono** - https://fonts.google.com/specimen/Ubuntu+Mono

## Або використай системний шрифт:

```bash
# Linux
cp /usr/share/fonts/truetype/dejavu/DejaVuSans.ttf fonts/default.ttf

# Або будь-який інший TTF файл
```

## Формат:
- Файл: `fonts/default.ttf`
- Тип: TrueType Font (.ttf)
- Налаштування в `config.json` → `ui.font`
